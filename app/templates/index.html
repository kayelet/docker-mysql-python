Welcome to my Python-Mysql platform!

How to run the SQL files

From container (from /app): python main.py

From terminal: docker exec -ti mysql_flask_app python main.py

To run a specific SQL: docker exec -ti mysql_flask_app python main.py <SQL>

To run SQLs from a directory: docker exec -ti mysql_flask_app python main.py <dir path to SQLs>

** Note: SQLs default path is /tmp if running SQL/s from a different location in container, it must be specified.

Navigate according the options below

  • To get failed SQLs: http://localhost:5000/failed
  • To get success SQLs: http://localhost:5000/success
  • To get the LOG record for a specific SQL run_id: http://localhost:5000/run_id?id=<run_id>
  • To update a status of an SQL to SUCCESS: http://localhost:5000/update-to-success?id=<run_id>
  • To drop all tables, LOG table included: http://localhost:5000/drop_tables
